h2. Setup and run

h3. Install:

* Ruby 2.5.1
* RubyGems 2.7.7
* Bundler 1.16.2
* PostgreSQL 9.6 (some features will not work in other db)
* nginx
* gettext
* curl
* sphinx search engine
* node.js
* /usr/bin/md5sum from coreutils
* git
* memcached
* redis
* graphviz
* rpm
* rpm2cpio
* bzip2
* GNU coreutils
* GNU cpio

or with the single line:

    apt-get install postgresql9.6-server postgresql9.6-contrib postgresql9.6 bzip2 \
                    gettext graphviz memcached /usr/bin/md5sum npm nodejs curl sphinx \
                    libruby-devel ruby redis zlib-devel postgresql9.6-devel

h3. Then:

* @bundle install@
* @service postgresql start@
* @service redis start@
* @memcached -d -m 128@ (as a user)

* @bin/rake secret@
* @bin/rake gettext:pack@

h3. Init database:

* @bundle exec rails db:create@
* @bundle exec rails db:migrate@
* @bundle exec rails db:seed@

h3. Import data:

* @bundle exec rake sisyphus:update@

* @bundle exec rake sisyphus:bugs@
* @bundle exec rake sisyphus:repocops@
* @bundle exec rake sisyphus:repocop_patches@
* @bundle exec rake sisyphus:sisyphus:update_repocop_cache@
* @bundle exec rake gear:import@
* @bundle exec rake perlwatch:update@
* @bundle exec rake ftbfs:update@

* @bundle exec rake sisyphusarm:srpms@
* @bundle exec rake sisyphusarm:binary@

* @bundle exec rake update:branches@

* @bundle exec rake redis:cache@

h2. Deploy

* @bundle exec cap production deploy@

h2. Run on production

* @bundle exec rake assets:precompile@
* @redis-server redis.conf@

h2. Tests

* @bundle exec rspec@
